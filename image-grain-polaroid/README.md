# Grain & Polaroid Lab

A small static image tool that matches the dark studio theme from the SVG dither project.

## Use

Open `index.html` in a browser, upload an image, then apply:

- Grainy noise independently
- Polaroid-style framing independently
- Adjustable side border thickness, bottom lip, paper color, and tilt
- Simple tone controls for exposure, contrast, fade, and warmth

Use **Download PNG** to export the rendered canvas.
